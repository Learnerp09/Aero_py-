# 🚀 AeroPy

Python library for aerospace simulations, orbital mechanics, and rocket flight physics.

---

## 📌 About

AeroPy is my personal project to learn Python while exploring aerospace engineering concepts through simulations and visualizations.

The goal is to gradually build a collection of aerospace-related modules instead of isolated scripts.

---

## 🎯 Goals

- Learn Python through real engineering projects
- Build accurate physics simulations
- Practice writing clean, maintainable code
- Create visualizations of aerospace concepts
